# Roadmap

## v1.0 (current)

- [x] First-run wizard
- [x] Tab-based control panel
- [x] Full CLI (`analyze`, `cleanup`, `dry-run`, `restore`, `update`, `list-languages`)
- [x] Shared backend library
- [x] YAML-driven language database (50+ languages)
- [x] apt simulation safety gate
- [x] JSON rollback/restore
- [x] Docker CI environment
- [x] GitHub Actions pipeline
- [x] Debian packaging
